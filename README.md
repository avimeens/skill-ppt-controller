## Ppt Controller
Control MS ppt presentation

## Description
Use voice to open, close, advance slides (next slide, prev slide, last slide) from a user's folder

## Examples
 * "open presentation"
 * "next slide"
 * "previous slide"
 * "first slide"
 * "last slide"
 * "close presentation"
 * "open file"
 * "close file"
